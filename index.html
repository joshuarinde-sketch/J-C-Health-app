📋 Firebase Setup Guide

Follow these steps — takes about 5 minutes. All free!

  1. Go to console.firebase.google.com and sign in with your Google account.
  2. Click "Add project" → name it "jc-fitness" → click through the prompts → Create project.
  3. On the project dashboard, click the Web icon (</>) to add a web app.
  4. Name it anything (e.g. "jc-app") → click Register app.
  5. You'll see a config block. Copy the values into this app:
    • apiKey
    • authDomain
    • projectId
    • appId
  6. Back in Firebase, go to Build → Firestore Database → Create database → choose Start in test mode → select a region → Done.
  7. That's it! Paste your config values into this app and hit Save.
⚡ Both phones need the same config. The easiest way: set it up on one phone, then text the 4 values to the other person.
